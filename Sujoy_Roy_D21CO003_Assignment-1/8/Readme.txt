The program is performed on Windows.

To compile the program type:- gcc 8_binary_avl.c -o 8_binary_avl 
To run the program type:- ./8_binary_avl.out

We have taken 10000 as input to the tree.
Time to insert in binary tree is:- 0.004623 seconds
Time to insert in AVL tree is :- 0.008324 seconds

Time needed to search the item in binary tree is : 0.000050 seconds
Time needed to search the item in AVL tree is : 0.000034 seconds

Search time in AVL tree is lesser than binary tree.
