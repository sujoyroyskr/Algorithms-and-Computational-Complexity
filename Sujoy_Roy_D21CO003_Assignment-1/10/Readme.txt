The program is performed on Windows.
To compile the code type:- g++ Q10.cpp -o Q10
To run the code type :- ./Q10.out

If the user add more than 20 nodes then the program will break.
To add node type "add" "value" then enter
To extract type "extract"
To do nothing type "neither"
To find min heap type "min"
To find max heap type "max"
To sort type "sort"
To end type "end"
